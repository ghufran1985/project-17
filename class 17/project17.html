pagination

    pagination: A Bootstrap class that styles a list as a pagination component.
    It is typically used to create a series of links that allow users to navigate
    through multiple pages of content. The pagination class provides default styling
    for active, disabled, and hovered states, making it visually distinct and
    user-friendly.


    page-item: A Bootstrap class used within a pagination component to style each
    individual page link. It helps in creating a visually consistent list of
    pagination links, allowing for additional styling and functionality



    page-link: A Bootstrap class that styles a link as part of a pagination
    component. It provides the appropriate appearance for pagination links,
    including padding and hover effects, ensuring they are visually distinct
    and user-friendly. The href="#" is a placeholder for the actual link
    destination.
